# PRP for P1.M7.T2.S1

## Objective

Wire the stagehand CLI's main surface: cobra root command `Run` (the default action), all PRD ¬ß15.2 persistent global flags, precedence loading via `config.Flags` (CLI flags + STAGEHAND_* env), the default action flow (provider validation ‚Üí maybeAutoStage ‚Üí `stagehand.GenerateCommit` ‚Üí sentinel‚Üíexit-code mapping), --version short-circuit, and TTY-aware verbose/color. ## Context

## Goal

**Feature Goal**: The runnable default action ‚Äî `stagehand [flags]` resolves config (flag>env>git-config>file>defaults via config.Load), validates the provider exists+is-on-PATH (friendly error), runs maybeAutoStage (HasStagedChanges/AddAll per FR16‚ÄìFR20), calls `stagehand.GenerateCommit`, and maps the returned sentinel error to the exact PRD ¬ß15.4 exit code (0/1/2/3), with verbose (FR50) and TTY-aware color (FR51). `--version` short-circuits via cobra's built-in Version field. The ten PRD ¬ß15.2 global flags are registered as persistent flags on rootCmd.

**Deliverable**: (a) EDIT `cmd/stagehand/main.go` ‚Äî add the persistent flags, set `rootCmd.Run` (or RunE) wiring `os.Exit(runDefault(...))`, set `rootCmd.Long`; (b) CREATE `cmd/stagehand/run.go` ‚Äî owns the default-action logic as PURE, testable helpers (`buildFlags`, `resolveAndCheckProvider`, `maybeAutoStage`, `mapErrorToExitCode`, `runDefault`); (c) CREATE `cmd/stagehand/run_test.go` ‚Äî white-box `package main` tests for the MOCKING contract; (d) EDIT `docs/CONFIGURATION.md` ‚Äî add the ¬ß15.2 CLI flag table + the ¬ß15.4 exit-code table (Mode A). **Success Definition**: `go build ./cmd/stagehand/`, `go vet ./cmd/stagehand/`, gofmt clean, and `go test ./...` all pass; `stagehand --version` prints `stagehand version <v>` and exits 0; bare `stagehand` (in a repo with staged changes + a detected agent) runs the default action and exits 0; nothing-staged + `--no-auto-stage` exits 2; missing/undetected provider exits 1 with the friendly message; precedence flag>env is asserted by test.

## User Persona

**Target User**: a developer who has staged changes and wants an AI-generated conventional commit message.
**Use Case**: `stagehand` (bare) in a repo with staged changes; `stagehand --dry-run`; `stagehand --provider claude`; `stagehand --no-auto-stage`.
**User Journey**: invoke ‚Üí config resolved ‚Üí (trust notice if repo-local provider) ‚Üí (auto-stage if nothing staged & auto_stage_all) ‚Üí generate ‚Üí print `[sha] subject` + diff-tree ‚Üí exit 0 (or 2/3/1).
**Pain Points Addressed**: no need to remember a message format; transparent verbose mode; safe exit codes for scripting.

## What

### User-visible behavior
- `stagehand` (bare, staged changes present, agent detected): generates + commits, prints `[<short-sha>] <subject>` then `git diff-tree --no-commit-id --name-status -r <sha>` to stdout (FR42), exits 0.
- `stagehand --version`: prints `stagehand version <v>`, exits 0 (cobra Version field).
- `stagehand --dry-run`: prints the generated message to stdout, does NOT commit, exits 0 (FR49).
- `stagehand --no-auto-stage` with nothing staged: prints a friendly message to stderr, exits 2 (FR17).
- Nothing staged + auto_stage_all (default): runs `git add -A`, prints `Nothing staged ‚Äî staging all changes.` to stderr (FR18), re-checks; still clean ‚Üí exit 2 (FR17).
- `stagehand --all`/`-a`: force `git add -A` even when something is already staged (FR20).
- Undetected/unknown provider: `Provider <name>: command <cmd> not found. Is the agent installed?` to stderr, exit 1.
- Generation failure (timeout/agent-error/parse-fail/dup-exhaustion): rescue block printed to stderr by CommitStaged, exit 3 (ErrRescue).
- Repo-local provider redirect: one-line ¬ß19 trust notice to stderr before generation.
- `--verbose`/`-v`/`STAGEHAND_VERBOSE=1`: the resolved command, raw agent stdout, and each retry are printed to stderr (FR50) ‚Äî emitted by CommitStaged's Output.Verbosef, gated by cfg.Verbose which the CLI sets.
- Color auto-enables on a TTY stdout, disables with `--no-color`/`STAGEHAND_NO_COLOR`/`NO_COLOR` (FR51).

### Implementation Patterns & Key Details
```go
func main() {
    rootCmd.Run = func(cmd *cobra.Command, _ []string) { os.Exit(runDefault(cmd)) }
    if err := rootCmd.Execute(); err != nil { os.Exit(ui.ExitError) }
}
func runDefault(cmd *cobra.Command) int {
    flags := buildFlags(cmd)
    cfg, reg, notice, err := config.Load(flags, ".")
    out := ui.NewOutput(os.Stdout, os.Stderr, cfg.Verbose, cfg.NoColor)
    if err != nil { fmt.Fprintln(os.Stderr, err); return ui.ExitError }
    if notice != "" { out.Progressf("%s\n", notice) }   
    name, code, msg := resolveAndCheckProvider(cfg, reg)
    if code != 0 { out.Progressf("%s\n", msg); return code }
    g, err := git.New(".")
    if err != nil { fmt.Fprintln(os.Stderr, err); return ui.ExitError }
    allFlag, _ := cmd.Flags().GetBool("all")
    noAutoStage, _ := cmd.Flags().GetBool("no-auto-stage")
    if code := maybeAutoStagePolicy(g, out, cfg, allFlag, noAutoStage); code != 0 { return code }
    dryRun, _ := cmd.Flags().GetBool("dry-run")
    opts := stagehand.Options{Provider: cfg.Provider, Model: cfg.Model, Timeout: cfg.Timeout, DryRun: dryRun}
    _, err = stagehand.GenerateCommit(context.Background(), opts)
    return mapErrorToExitCode(err)   
}
func buildFlags(cmd *cobra.Command) config.Flags {
    f := config.Flags{}
    if v, ok := os.LookupEnv("STAGEHAND_PROVIDER"); ok && v != "" { f.Env.Provider = &v }
    if cmd.Flags().Changed("provider") { v, _ := cmd.Flags().GetString("provider"); f.Flag.Provider = &v }
    return f
}
```
