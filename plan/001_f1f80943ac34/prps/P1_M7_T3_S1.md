# PRP for P1.M7.T3.S1

## Objective

Add the stagehand providers command tree (FR46–FR48, US10) in a new cmd/stagehand/providers.go (package main): a `providers list` subcommand that resolves config + registry via config.Load, prints every built-in+user provider name (reg.List), marks each detected-on-$PATH vs not (reg.Detect), and shows the resolved default provider+model; and a `providers show <name>` subcommand that prints the fully-resolved manifest (built-in field-merged with user overrides via the registry) as TOML using the Manifest's existing toml tags. Register the tree onto rootCmd via a package-level init() so main.go is untouched (zero conflict with sibling P1.M7.T2.S1). Consume provider.Registry (P1.M2.T3.S2) and config.Load (P1.M5.T3.S1) read-only; add cmd/stagehand/providers_test.go covering the MOCKING contract (builtins detected + fabricated not-detected; show TOML round-trips to the same manifest with nil-vs-empty normalization; a user override is reflected in show output) and create docs/PROVIDERS.md as the Mode A provider reference surface.

**Feature Goal**: Add the `stagehand providers` command tree (FR46–FR48, US10) — a `providers list` subcommand that prints every built-in + user provider, marks each detected-on-$PATH vs not, and shows the resolved default provider+model; and a `providers show <name>` subcommand that prints a provider's FULLY-RESOLVED manifest (built-in field-merged with user overrides) as TOML so users can debug exact command rendering.

**Deliverable**:

Includes pure, hermetic render helpers (`renderProvidersList`, default-resolution) and a thin `show` path that encodes a `provider.Manifest` as TOML.
2. `cmd/stagehand/providers_test.go` (package main) — white-box tests covering the MOCKING contract: list marks builtins detected (real PATH) + a fabricated name not-detected and shows the default; show emits TOML that decodes back to the same manifest (nil-vs-empty normalized); a user override is reflected in show output.

**Success Definition**: `go test ./cmd/stagehand/ -run TestProviders -v` passes; `go run ./cmd/stagehand providers show pi` prints valid TOML that decodes to pi's built-in manifest; `go run ./cmd/stagehand providers list` marks `pi` detected and a fabricated provider not-detected, and prints a `default provider:` line; `gofmt`/`go vet`/`go build`/`go test ./...` are all clean.

## User Persona

**Target User**: a developer / new-user evaluator (US10) who runs several coding agents.
**Use Case**: before committing, understand which agents stagehand can drive, which are actually installed, and which one is the resolved default — and debug the exact command line stagehand will render for a given provider.
**User Journey**: `stagehand providers list` → see installed agents + default → `stagehand providers show pi` → see the full merged manifest as TOML → tweak `[provider.pi]` in config → re-run `show` to confirm the override took.
**Pain Points Addressed**: "which agent will actually run?" (US10) and "why is stagehand invoking the agent THIS way?" (manifest debug surface, PRD §9 risk mitigation line 1292).

## Why
- **Discovery UX (US10)**: `providers list` is the first thing a new user runs to see what is installed and what the default is.
- **Debugging surface**: `providers show <name>` emits the EXACT merged manifest (built-in ⊕ user override) as TOML, so a user can verify command rendering without reading Go source (PRD §9 risk row "Agent CLI surfaces change").
- **Wires already-built dependencies**: consumes `provider.Registry` (P1.M2.T3.S2: List/Detect/Get) and `config.Load` (P1.M5.T3.S1: resolved default + override-merged registry). This task is pure CLI glue + rendering — no new domain logic.

## What

### `stagehand providers list` (FR46)
- Resolve `cfg, reg, _, err := config.Load(config.Flags{}, "")` (repoDir `""` = inherit cwd per readRepoFile). Ignore `trustNotice` (only the commit action prints it). Propagate `err` (e.g. malformed global config) → exit 1.
- `detected := reg.Detect()`. For each name in `reg.List()` (already sorted), print a row: name, a `(default)` marker on the default provider, status `detected`/`not detected`, and the manifest's `DefaultModel` (or `(unset)` if empty).
- Resolve the default: `defaultName = cfg.Provider`; if `""`, the first detected provider in `reg.List()` order (the auto-detected default). `defaultModel = cfg.Model`; if `""`, the defaultName manifest's `DefaultModel`. If no provider is detected at all, `defaultName = ""` → print `(none detected)`.
- Print a trailing `default provider: <name> (model: <model>)` line.
- Uses `cobra.NoArgs`.

### `stagehand providers show <name>` (FR47)
- `cobra.ExactArgs(1)`. Resolve `reg` via `config.Load(config.Flags{}, "")`.
- `m, ok := reg.Get(args[0])`; if `!ok` → return `fmt.Errorf("unknown provider %q: run 'stagehand providers list' to see available providers", args[0])` (exit 1).
- Print the FULLY-RESOLVED manifest as TOML: `toml.NewEncoder(cmd.OutOrStdout()).Encode(m)`. The `Manifest` struct already carries `toml:` tags, so no DTO is needed.

### Success Criteria
- [ ] `providers list` prints all six built-ins (pi, claude, gemini, opencode, codex, cursor) plus any user providers, each marked `detected` or `not detected`, and a `default provider:` line.
- [ ] `providers show <name>` prints valid TOML for a known provider and errors (exit 1) for an unknown one.
- [ ] A user override (e.g. `[provider.pi] default_model = "x"`) is reflected in `show pi` output.
- [ ] The command tree is registered without modifying `main.go`.

func renderProvidersList(w io.Writer, reg *provider.Registry, detected map[string]bool, defaultName, defaultModel string) error {
    fmt.Fprintln(w, "PROVIDER   STATUS         DEFAULT MODEL")
    for _, name := range reg.List() {               
        status := "not detected"
        if detected[name] { status = "detected" }
        m, _ := reg.Get(name)
        marker := ""
        if name == defaultName { marker = " (default)" }
        model := m.DefaultModel
        if model == "" { model = "(unset)" }
        fmt.Fprintf(w, "%-14s %-14s %s%s\n", name, status, model, marker)
    }
    dn, dm := defaultName, defaultModel
    if dn == "" { dn = "(none detected)" }
    if dm == "" { dm = "(unset)" }
    fmt.Fprintf(w, "\ndefault provider: %s (model: %s)\n", dn, dm)
    return nil
}
func manifestEqual(a, b provider.Manifest) bool {
    if len(a.Subcommand) == 0 && len(b.Subcommand) == 0 { a.Subcommand, b.Subcommand = nil, nil }
    if len(a.BareFlags) == 0 && len(b.BareFlags) == 0 { a.BareFlags, b.BareFlags = nil, nil }
    if len(a.Env) == 0 && len(b.Env) == 0 { a.Env, b.Env = nil, nil }
    return reflect.DeepEqual(a, b)
}
