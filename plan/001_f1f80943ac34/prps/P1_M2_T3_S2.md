# PRP for P1.M2.T3.S2

## Objective

Create internal/provider/registry.go: a Registry that precomputes a field-by-field merge of user provider overrides onto the six built-in manifests (decisions.md §6), exposes Get/List, and detects installed agents on $PATH via exec.LookPath (PRD FR46-FR48). Consumed by config Load (P1.M5.T3.S1) and CLI providers list/show (P1.M7.T3.S1). No import of internal/config (constructor-injected overrides) — preserves the no-import-cycle milestone invariant.

## Context

## Goal

**Feature Goal**: A `provider.Registry` that merges user provider overrides field-by-field over the six built-in manifests and detects which providers are installed on `$PATH`, with Get/List/Detect methods consumed by the config loader and the CLI `providers` subcommands.

**Deliverable**: `internal/provider/registry.go` (plain `package provider`) plus `internal/provider/registry_test.go`, defining `Registry`, `NewRegistry(builtins, overrides map[string]Manifest) *Registry`, `Get(name string) (Manifest, bool)`, `List() []string`, and `Detect() map[string]bool`.

**Success Definition**: `go test ./internal/provider/ -run TestRegistry -v` passes, covering the four contract scenarios (override default_model keeps BareFlags; override BareFlags replaces; new name resolves; detect true for real `pi` and false for a bogus command). `go vet`, `gofmt -l`, and the full `go test ./...` are green. `registry.go` imports only `os/exec` and `sort` (NO `internal/config`).

## Why

- **Config precedence (decisions.md §6)**: provider manifests merge FIELD-BY-FIELD — a user override setting only `default_model` keeps every other built-in field. This is the merge engine behind config Load.
- **FR46-FR48**: `providers list` (List + Detect), `providers show <name>` (Get, fully-resolved manifest), and user-defined providers that override built-ins or add new names (FR48).
- **No import cycle (plan_overview key decision 1)**: `provider/registry` must NOT import `internal/config`. Overrides are constructor-injected as `map[string]Manifest`. `config` (M5) imports `provider`; never the reverse.

### Implementation Tasks (ordered by dependencies)
```yaml
Task 1: CREATE internal/provider/registry.go — types + constructor
  - FILE HEAD: plain `package provider` (NO doc comment block; package doc lives in manifest.go).
  - IMPORTS: only "os/exec" and "sort". Do NOT import internal/config.
  - IMPLEMENT: type Registry struct { merged map[string]Manifest }
  - IMPLEMENT: func NewRegistry(builtins, overrides map[string]Manifest) *Registry
      1. merged := make(map[string]Manifest, len(builtins)+len(overrides))
      2. for name, m := range builtins { merged[name] = cloneManifest(m) }   
      3. for name, ov := range overrides {
             if base, ok := merged[name]; ok { merged[name] = mergeManifest(base, ov) } 
             else { merged[name] = cloneManifest(ov) }                                  
         }
      4. return &Registry{merged: merged}
  - DOC: cite decisions.md §6 (field-by-field) and plan_overview key decision 1 (no config import).
Task 2: IMPLEMENT helpers mergeManifest + cloneManifest (same file)
  - func cloneManifest(m Manifest) Manifest: deep-copy Subcommand & BareFlags via append([]string(nil), s...)
    and Env via a fresh map[string]string; return the value copy. Guards registry against caller aliasing.
  - func mergeManifest(base, ov Manifest) Manifest:
      out := cloneManifest(base)
      for each of {Detect, Command, PromptDelivery, PromptFlag, PrintFlag, ModelFlag,
                    DefaultModel, SystemPromptFlag, ProviderFlag, DefaultProvider, Output,
                    JSONField, RetryInstruction}: if ov.<f> != "" { out.<f> = ov.<f> }
      if ov.Name != "" { out.Name = ov.Name }
      if len(ov.Subcommand) > 0 { out.Subcommand = append([]string(nil), ov.Subcommand...) }
      if len(ov.BareFlags) > 0  { out.BareFlags  = append([]string(nil), ov.BareFlags...) }
      if len(ov.Env) > 0 { ne := make(map[string]string, len(ov.Env)); for k,v := range ov.Env { ne[k]=v }; out.Env = ne }
      if ov.StripCodeFence { out.StripCodeFence = true }
      return out
  - GOTCHA note in a comment: bool cannot be flipped false-over-true via merge (documented v1 limitation).
Task 3: IMPLEMENT Get / List / Detect (same file)
  - func (r *Registry) Get(name string) (Manifest, bool): return r.merged[name], ok  (ok=false for unknown)
  - func (r *Registry) List() []string: collect keys, sort.Strings, return. (FR46 providers list, stable order)
  - func (r *Registry) Detect() map[string]bool:
      result := make(map[string]bool, len(r.merged))
      for name, m := range r.merged {
          target := m.Detect; if target == "" { target = m.Command }
          if target == "" { result[name] = false; continue }
          _, err := exec.LookPath(target)
          result[name] = (err == nil)
      }
      return result
  - DOC: cite FR46 (list+detect), FR47 (show via Get), FR48 (override/new-name).
Task 4: CREATE internal/provider/registry_test.go — package provider (white-box)
  - IMPORTS: testing, reflect, sort, and the local provider pkg (white-box, same pkg).
  - Mirror builtin_test.go conventions: inline Manifest literals, reflect.DeepEqual, t.Run subtests.
  - REQUIRED test cases (the 4 contract MOCKING scenarios) — these are acceptance-critical:
      TestRegistry_MergeDefaultModelKeepsBareFlags:
          r := NewRegistry(Builtins(), map[string]Manifest{"pi": {DefaultModel: "x"}})
          got, ok := r.Get("pi"); require ok && got.DefaultModel=="x"
          assert reflect.DeepEqual(got.BareFlags, Builtins()["pi"].BareFlags)
      TestRegistry_MergeBareFlagsReplaces:
          repl := []string{"--only-this"}
          r := NewRegistry(Builtins(), map[string]Manifest{"pi": {BareFlags: repl}})
          got, _ := r.Get("pi")
          assert reflect.DeepEqual(got.BareFlags, repl)            
          assert !reflect.DeepEqual(got.BareFlags, Builtins()["pi"].BareFlags)
          repl[0] = "--mutated"; assert got.BareFlags[0] == "--only-this"
      TestRegistry_NewNameResolves:
          r := NewRegistry(Builtins(), map[string]Manifest{"myagent": {Command: "/opt/myagent/bin/agent", PromptDelivery: "stdin"}})
          got, ok := r.Get("myagent"); assert ok && got.Command == "/opt/myagent/bin/agent"
          assert contains(r.List(), "myagent")
      TestRegistry_Detect:
          r := NewRegistry(Builtins(), map[string]Manifest{"definitely-not-an-agent-xyz": {Command: "definitely-not-an-agent-xyz"}})
          d := r.Detect()
          assert d["pi"] == true                                   
          assert d["definitely-not-an-agent-xyz"] == false         
  - ADDITIONAL coverage:
      TestRegistry_GetUnknownReturnsFalse: Get("nope") → (Manifest{}, false)
      TestRegistry_ListSortedAndUnion: List() is sorted and contains exactly the 6 builtins + any overrides (no dupes)
      TestRegistry_EnvReplaceWholesale: override{Env:{"X":"1"}} on a builtin with nil Env → merged Env=={"X":"1"};
          verify it is a COPY (mutating override Env after does not change Get result)
      TestRegistry_NewRegistryDoesNotMutateBuiltins: build a registry with overrides, then confirm a fresh
          Builtins()["pi"] is unchanged (registry must not mutate S1's maps/slices)
      TestRegistry_DetectUsesCommandWhenDetectEmpty: provider with Detect="" and Command="pi" → detected true
  - HELPERS: reuse the `contains` helper pattern from builtin_test.go (define locally if not exported).
Task 5: VALIDATE — gofmt, go vet, go test ./...
  - See validationGates. Fix any failure before declaring done.

```

## Implementation Steps

1. CREATE internal/provider/registry.go with plain `package provider`; imports ONLY os/exec and sort (NO internal/config — plan_overview key decision 1). Define `type Registry struct { merged map[string]Manifest }`.

4. IMPLEMENT mergeManifest(base, ov Manifest) Manifest: clone base, then apply each NON-EMPTY scalar override string (Detect, Command, PromptDelivery, PromptFlag, PrintFlag, ModelFlag, DefaultModel, SystemPromptFlag, ProviderFlag, DefaultProvider, Output, JSONField, RetryInstruction, and Name if set), replace Subcommand/BareFlags WHOLESALE when len>0, replace Env WHOLESALE (not deep-merged) when len>0, and set StripCodeFence only when ov.StripCodeFence==true. Document the bool zero-value limitation in a comment (decisions.md §6).
5. IMPLEMENT Get(name string) (Manifest, bool) returning r.merged[name] and ok (ok=false for unknown — used by FR47 providers show and M5 config Load).
6. IMPLEMENT List() []string collecting r.merged keys then sort.Strings (FR46 providers list, deterministic order).
7. IMPLEMENT Detect() map[string]bool: for each name use the merged manifest's Detect, or Command if Detect empty, then exec.LookPath; result[name] = (err==nil); false when both Detect and Command empty (FR46 detect).
8. CREATE internal/provider/registry_test.go (white-box `package provider`) mirroring builtin_test.go conventions (reflect.DeepEqual, inline literals, t.Run).
