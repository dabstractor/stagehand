## Why

- This is the **Mode B changeset-level doc** — the marketing surface the whole v1.0 release is judged by. Every per-file Mode A doc slice written throughout the milestone (docs/CONFIGURATION.md, docs/PROVIDERS.md, the install.sh §21.3 docstring) points at this README as the entry point; it must exist and be accurate.
- PRD §21.5 prescribes the exact section order; deviating loses the narrative arc (pitch -> demo -> differentiation -> install -> use -> configure -> the payoff diagram -> reference -> contribute -> who-it's-not-for).
