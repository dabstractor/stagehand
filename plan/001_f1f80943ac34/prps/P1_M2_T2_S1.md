## Why

- **The robustness centerpiece of output handling.** PRD §17.4 (decisions D1) deliberately switched from commit-pi's fragile JSON+sed contract to a RAW default ("output only the message") because raw has nothing to escape/parse and removes the "no double quotes in the message" constraint. §12.9 is the cleanup pipeline that makes raw reliable: it tolerates a model wrapping output in a code fence, tolerates preamble around JSON, and NEVER hard-fails on a parse glitch — it falls back to the cleaned raw stdout.
- **Pure + table-testable.** parseOutput takes a string + a Manifest and returns (string, bool); it does no I/O, imports no os/exec/log/config. That makes the full behavior verifiable with a byte-exact table — no process spawning. (Verified by a throwaway prototype: all 12 contract + edge cases pass; see research note §6.)
- **Consumed by the generate parse-correction loop (P1.M6.T1.S1).** The inner loop calls parseOutput on agent stdout; `ok==false` triggers one corrective retry; `ok==true` yields the message. Getting this byte-exact now unblocks M6 and the executor (M2.T4.S1).

### Implementation Patterns

```go
func parseOutput(raw string, m Manifest) (string, bool) {
	// (1) trim
	s := strings.TrimSpace(raw)

	// (2) optional single-layer code-fence unwrap (§12.9 step 2)
	if m.StripCodeFence && (strings.HasPrefix(s, "```") || strings.HasPrefix(s, "~~~")) {
		fence := s[:3] // ``` or ~~~ — matches only its own kind
		if nl := strings.IndexByte(s, '\n'); nl != -1 {
			s = s[nl+1:] // drop opener line incl. any language tag
		} else {
			s = "" // opener was the entire string
		}
		if last := strings.LastIndex(s, fence); last != -1 {
			s = s[:last] // drop everything from the LAST closer onward
		}
		s = strings.TrimSpace(s)
	}

	// (3) output switch — default ("") and "raw" → msg=s; "json" → extract w/ raw fallback
	msg := s
	if m.Output == OutputJSON {
		if v, ok := extractJSON(s, m.JSONField); ok {
			msg = v
		} // else: fall back to raw msg=s (§17.4)
	}

	// (4) normalize newlines: \r\n→\n first, then collapse 3+\n→\n\n
	msg = strings.ReplaceAll(msg, "\r\n", "\n")
	msg = collapseNewlines.ReplaceAllString(msg, "\n\n")

	// (5) final trim + ok
	msg = strings.TrimSpace(msg)
	return msg, msg != ""
}

func extractJSON(s, field string) (string, bool) {
	var obj map[string]any
	if err := json.Unmarshal([]byte(s), &obj); err == nil {
		if v, ok := obj[field].(string); ok {
			return v, true
		}
	}
	// balanced-substring retry: first '{' … last '}'
	if start := strings.IndexByte(s, '{'); start != -1 {
		if end := strings.LastIndexByte(s, '}'); end != -1 && end > start {
			var obj2 map[string]any
			if err := json.Unmarshal([]byte(s[start:end+1]), &obj2); err == nil {
				if v, ok := obj2[field].(string); ok {
					return v, true
				}
			}
		}
	}
	return "", false
}
```
